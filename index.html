General Questions

What is JavaScript and why it is used ?

JavaScript is a text-based programming language used both on the client-side and server-side that allows you to make web pages interactive. JavaScript is mainly used for web-based applications and web browsers.

What is DOM and it's importance ?

The way a document content is accessed and modified is called the Document Object Model. It allows a language (JavaScript) to manipulate, structure, and style your website.

Can anyone tell me the examples where DOM is used ?

Some examples are:- 1] ColorPicker - We can change the color dynamically.
2] Toggling of a Button - On click, button changes it's functionality. and many more.
